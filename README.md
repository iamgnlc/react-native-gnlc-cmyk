# react-native-gnlc-cmyk

![last commit](https://img.shields.io/github/last-commit/iamgnlc/react-native-gnlc-cmyk)

React Native demo app

## Install Expo

```
$ yarn global add expo-cli
```

## Install dependencies

```
$ yarn install
```

## Run in simulator

```
$ yarn ios

$ yarn android

$ yarn web
```
